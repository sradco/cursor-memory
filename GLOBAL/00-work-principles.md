# Work Principles

## Core Values

1. **Context is king** — Every decision must be traceable to a recorded rationale.
2. **Topic isolation** — One topic per conversation thread; never mix concerns.
3. **Summaries over transcripts** — Distill conversations into actionable notes, not copy-paste logs.
4. **Durable memory** — If it's not written down, it didn't happen. Files survive; chat history does not.
5. **Small, frequent commits** — Summarize at the end of every session, not once a week.

## How to Use This File

- Review before starting a new project or topic.
- Update when the team agrees on a new principle.
- Reference in `decisions.md` when a principle drives a choice.
